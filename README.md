# 🛍️ Marketplace (Spring Boot + SwiftUI)

Este é um projeto de um **Marketplace completo**, desenvolvido do zero com backend em **Spring Boot** e frontend em **SwiftUI**.

## 🚀 Funcionalidades (em desenvolvimento)

- Cadastro de usuários
- Cadastro de produtos
- Listagem de produtos
- Compra simulada
- Integração com app iOS

## 🛠️ Tecnologias

- Java 17
- Spring Boot
- PostgreSQL
- SwiftUI
- Git e GitHub
